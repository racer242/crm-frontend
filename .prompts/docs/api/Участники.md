| :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| POST Content-Type: multipart/form-data                                                                                                                                                           |
| { }                                                                                                                                                                                              |
| Тело файла                                                                                                                                                                                       |
