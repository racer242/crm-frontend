Ознакомься с эндпойнтом получения данных пользователя.
.prompts\docs\api\Данные участника.md

Конфигурация страницы вывода данных пользователя должна быть здесь: config\pages\user-acts.json

На нее осуществляется переход в виде http://host/users/23747/acts
где 23747 - это id пользователя.
В рамках конфигурации страницы, его можно получить через макрос location (изучи как).

Запросы осуществляеются через свойство dataFeed (серверная загрузка данных перед серверным рендером, см docs\data-feed-reference.md), таким же образом, как на странице config\pages\users.json (список актов), config\pages\user-docs.json

Визуализация должна быть построена на компонентах системы docs\components-reference.md

На странице должен быть список актов пользователя и кнопка "Добавить" которая ведет на роут /acts/add
