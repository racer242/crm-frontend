Конфигурация страницы вывода данных пользователя должна быть здесь: config\pages\user-docs.json

На нее осуществляется переход в виде http://host/users/23747/docs
где 23747 - это id пользователя.
В рамках конфигурации страницы, его можно получить через макрос location (изучи как).

Запрос осуществляется через свойство dataFeed (серверная загрузка данных перед серверным рендером, см docs\data-feed-reference.md), таким же образом, как на странице config\pages\user.json (без изменений)

Визуализация должна быть построена на компонентах системы docs\components-reference.md
