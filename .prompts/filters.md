# Фильтрация списка.

Это решается универсальным подходом.

В рамках UI системы мы реализуем набор компонентов:

1. ## Панель фильтров - на основе Drawer.

Активируется по изменению состояния линкуемого свойства в state типа Boolean. Имя state-свойства задается в настройках props компонента через путь “@ЭЛЕМЕНТ.state.свойство-переключатель” (или “state.свойство-переключатель” для корневого sate страницы). На странице будет присутствовать компонент (обычно кнопка), который по событию onClick в обработчике будет менять свойство ЭЛЕМЕНТ.state.свойство-переключатель через команду toggleProperty.

Панель, при активации (параметр = true), выезжает (с какой стороны - передается через props для Drawer).

Список фильтров скроллится, если он большое высоты окна.

Внизу панели - фиксированная панель, на которой кнопки - применить, очистить.

Данные компонента передаются через свойство value, которое хранит данные фильтра в формате модели (рассмотрена ниже). В обычной ситуации, это линковка на свойство в state - “@ЭЛЕМЕНТ.state.свойство-фильтры”. 

При применении или удалении фильтров, вызывается событие onChange, несущее в value текущую модель фильтров. Обработчик такого события сможет установить value в нужное свойство state “ЭЛЕМЕНТ.state.свойство-фильтры” через команду setState или mergeState.



2. ## Панель, отображающая текущие фильтры. Это горизонтальная панель, на которой каждое значение фильтра выглядит в виде компонента-чипа. Чипы располагаются в горизонтальном порядке, с wrapping.

Панель выводится только есть есть выбранные фильтры. Если фильтров нет - панель не выводится совсем.

Надписи чипов выводятся по следующему принципу исходя из типа фильтра:

Чекбокс, Переключатель - просто название фильтра.

Текстовый ввод, Числовой ввод, Дата, Выбор из вариантов - “Название фильтра: Значение”

Диапазон  - “Название фильтра: Начало – Конец”

Период дат - “Название фильтра: Дата начала – Дата конца”

Набор вариантов - просто название каждого варианта добавляется как чип.

Каждый чип имеет кнопку “закрыть”. При закрытии, чип удаляется и генерируется событие onChange.

На панели, слева есть кнопка “Очистить”, которая удаляет все фильтры.

Данные компонента передаются через свойство value, которое хранит данные фильтра в формате модели (рассмотрена ниже). В обычной ситуации, это линковка на свойство в state - “@ЭЛЕМЕНТ.state.свойство-фильтры”. 

При удалении фильтров, вызывается событие onChange, несущее в value текущую модель фильтров. Обработчик такого события сможет установить value в нужное свойство state “ЭЛЕМЕНТ.state.свойство-фильтры” через команду setState или mergeState.



## Модель данных фильтров:

[

  { “id”:”id1”, “name”:”Название 1”, “type”:”number”, “opts”:..., “value”:... }

  …

]



Где параметры фильтров:

| Параметр | Описание                   |
| -------- | -------------------------- |
| id       | Идентификатор фильтра      |
| name     | Название фильтра           |
| type     | Тип фильтра                |
| value    | Выбранное/текущее значение |
| opts     | Опции, варианты выбора     |



Типы фильтров:

| Компонент          | type     | Описание                            | opts                          | value                                   |
| ------------------ | -------- | ----------------------------------- | ----------------------------- | --------------------------------------- |
| Чекбокс            | checkbox | Переключает bool значение           |                               | true/false                              |
| Переключатель      | switch   | Переключает bool значение           |                               | true/false                              |
| Диапазон           | range    | Выбор из диапазона чисел            | [min,max]                     | [begin,end] |
| Слайдер            | slider   | Регулируемое слайдером значение     | [min,max]                     | 123                                     |
| Текстовый ввод     | text     | Ввод строки                         |                               | Текст                                   |
| Числовой ввод      | number   | Ввод числа                          |                               | 123                                     |
| Дата               | date     | Выбор даты                          |                               | 01-06-2026                              |
| Период дат         | period   | Выбор периода дат                   |                               | [01-06-2026,10-06-2026]                 |
| Набор вариантов    | options  | Выбор из списка bool значений       | [“Вариант 1”, “Вариант 2”...] | [true,false,...]                        |
| Выбор из вариантов | radio    | Выбор одного значения из нескольких | [“Вариант 1”, “Вариант 2”...] | Вариант 2                               |
