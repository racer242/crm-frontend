Инструкции для создании нового проекта:
Файлы необходимо размещать в корневой директории проекта. 
Создай файл .gitignore и пропиши туда все правила, соответствующие контексту используемого стека.
Создай файл .env для указания необходимых настроек проекта и предусмотри его применение в приложении.
Добавь в .env первые настройки, если они появились.
Предусмотри при старте серверной части приложения вывод баннера с основными параметрами конфигурации 
Добавь первые юнит-тесты и проверь их исполнение.
Сделай в readme.md описание функционала.
Сделай файл changelog.md краткое описание нового функционала и изменений.
Инициируй Git и сделай первый коммит.
